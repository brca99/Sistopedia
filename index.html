Windows 7

Windows 7 es un sistema operativo desarrollado por Microsoft, lanzado en el año 2009 como sucesor de Windows Vista. Fue muy popular debido a su mejor rendimiento, estabilidad y facilidad de uso, corrigiendo muchos problemas que tenía Vista.

Caracteristicas:

  • Barra de tareas mas practica
  • Mayor velocidad
  • Mayor Compatibilidad
  • Efectos visuales
  • Funciones Aero Snap y Aera Peek

Windows 8

Windows 8 es un sistema operativo desarrollado por Microsoft, lanzado en 2012 como sucesor de Windows 7. Fue diseñado pensando en dispositivos con pantalla táctil, como tablets, además de computadoras tradicionales.

Caracteristicas:

  • Optimizado para pantallas tactiles
  • Arranque mas veloz que Windows 7
  • Inclusion de Windows Store
  • Integracion con la nube
